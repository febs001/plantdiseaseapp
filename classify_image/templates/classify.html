ABOUT

The Inception v3 architecture uses centric transfer learning along with pre-configured model biases and weights for rapid feature extraction and compact multi-classification training. By manipulating a supplementary Imagenet feature extractor, a Softmax activation output node can be retrained with biases over over 1000 varied classes. This web application uses a combined Python Django + Tensorflow model instance to diagnose over 16 critical superficial blights and crop disease, frequently deleterious to agricultural output.

This project had been modified through utilizing existing open-source templates in order to implement plant disease detection in a user-friendly web application as part of Kaggle's PlantVillage dataset classification challenge.
